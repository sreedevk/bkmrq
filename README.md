# bkmrq
A Ruby Gem to Generate README from your browser bookmarks.

# Installation

Running Bkmrq requires ruby installed (>= 3.0.0)

```bash
  gem install bkmrq
```

# Usage

```bash
  bkmrq -H              # prints out help  
  bkmrq -B <browser>    # currently supported browsers - chromium | brave
```
